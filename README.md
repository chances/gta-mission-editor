# GTA Mission Editor

A web-based editor for creating 3D-era GTA mission scripts using [Blockly](https://developers.google.com/blockly).

## Usage

## Development

1. Install [Deno](https://deno.com/).
2. Clone this repository.
3. Start the development server:

   ```sh
   deno task dev
   ```
